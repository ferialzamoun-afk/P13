## 📌 **Blocs de Compétences RNCP 37837 Data Analyst (Option Statistiques)**

---

### **🔹 [RNCP37837BC01] Structurer et gérer la base de données**
   **Compétence** | **Description** | **Modalités d'évaluation** |
 |----------------|----------------|----------------------------|
 | **Créer une base de données** | Établir les tables et leurs relations en respectant les normes réglementaires pour une bonne gestion. | Projets + Soutenance |
 | **Gérer une base de données** | Établir des requêtes pour le remplissage/accès des tables, en respectant le RGPD. | Projets + Soutenance |
 | **Stratégie de requêtes SQL** | Extraire les données via SQL ou outils adaptés, en conformité RGPD, pour répondre à une problématique métier. | Projets + Soutenance |

---

### **🔹 [RNCP37837BC02] Identifier, collecter et analyser les données**
 | **Compétence** | **Description** | **Modalités d'évaluation** |
 |----------------|----------------|----------------------------|
 | **Identifier et collecter** | Utiliser des API, requêtes SQL ou outils adaptés pour préparer les données, en respectant les normes. | Projets + Soutenance |
 | **Extraire et agréger** | Techniques/outils pour structurer les données (définition des règles de nettoyage). | Projets + Soutenance |
 | **Explorer et pré-traiter** | Utiliser des langages/outils pour comprendre les caractéristiques des données. | Projets + Soutenance |
 | **Vérifier la cohérence** | Préparer et vérifier la fiabilité des données structurées. | Projets + Soutenance |
 | **Traiter les données manquantes** | Nettoyer et définir des règles de gestion (langages/outils), en conformité RGPD. | Projets + Soutenance |
 | **Analyse temporelle** | Explorer les données pour afficher la temporalité. | Projets + Soutenance |
 | **Analyse univariée/multivariée** | Décrire les données, détecter les incohérences, comprendre les corrélations. | Projets + Soutenance |

---

### **🔹 [RNCP37837BC03] Visualiser des données et interpréter des résultats**
 | **Compétence** | **Description** | **Modalités d'évaluation** |
 |----------------|----------------|----------------------------|
 | **Solution de visualisation** | Proposer une visualisation adaptée au public et au type de données (librairies/outils). | Projets + Soutenance |
 | **Graphiques accessibles** | Concevoir des graphiques (infographies) en respectant les bonnes pratiques d’accessibilité. | Projets + Soutenance |
 | **Créer un tableau de bord** | Utiliser des outils appropriés pour rendre la visualisation disponible. | Projets + Soutenance |
 | **Reporting des tendances** | Analyser les tendances des graphiques pour orienter les décisions stratégiques. | Projets + Soutenance |
 | **Récit des résultats** | Utiliser des procédés narratifs pour dynamiser la présentation. | Projets + Soutenance |
 | **Présenter les résultats** | Adapter le contenu au public, communication inclusive pour transmettre les informations clés. | Projets + Soutenance |

---

### **🔹 [RNCP37837BC04] Piloter un projet data en respectant la réglementation**
 | **Compétence** | **Description** | **Modalités d'évaluation** |
 |----------------|----------------|----------------------------|
 | **Veille métier/technologique** | Rechercher des solutions, tester de nouveaux outils/méthodes pour maintenir ses compétences à jour. | Projets + Soutenance |
 | **Expérimenter de nouvelles méthodes** | Tester des solutions identifiées via des recherches documentaires. | Projets + Soutenance |
 | **Identifier le besoin métier** | Intégrer les contraintes internes/externes pour définir les spécifications fonctionnelles. | Projets + Soutenance |
 | **Formaliser le cahier des charges** | Cadrer le besoin pour proposer une solution adaptée. | Projets + Soutenance |
 | **Organiser un projet data** | Utiliser des outils de gestion (rétroplanning) pour identifier temps/tâches/freins. | Projets + Soutenance |
 | **Ateliers de formation** | Former les équipes à la prise en main des outils. | Projets + Soutenance |
 | **Gérer la documentation** | Formaliser les processus pour une documentation fiable. | Projets + Soutenance |
 | **Adapter sa posture** | Se positionner comme consultant pour challenger les besoins. | Projets + Soutenance |

### **🔹 [RNCP37837BC05] Spécialisation Statistiques : Tests et Modèles d’apprentissage**

 | **Réduction de dimension** | Optimiser les temps d’apprentissage des modèles. | Projets + Soutenance |
 | **Tests statistiques** | Tester et valider des hypothèses sur une masse de données. | Projets + Soutenance |
 | **Feature Engineering (non supervisé)** | Sélectionner les variables pertinentes pour un apprentissage optimal. | Projets + Soutenance |
 | **Entraîner un modèle** | Supervisé/non supervisé via outils adaptés pour exploiter le modèle. | Projets + Soutenance |
 | **Exploiter un modèle** | Non supervisé (clustering) ou supervisé pour en apprendre davantage. | Projets + Soutenance |
 | **Choisir un modèle adapté** | Sélectionner le modèle adapté à la problématique métier pour une prédiction statistique. | Projets + Soutenance |
